# XE3 Poll
이 어플리케이션은 Xpressengine3(이하 XE3)의 플러그인입니다.

## 설명
플러그인을 설치하면 에디터에서 투표 기능을 사용할 수 있습니다.

주의) 댓글에서 투표 기능 사용하면 댓글이 동작되지 않습니다.
다음 버전부터 오류 없이 사용 가능합니다.
